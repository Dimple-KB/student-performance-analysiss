# 📊 Student Performance Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) to understand how
study hours and attendance affect student scores.

## Skills Used

-   Python
-   Pandas
-   Data Visualization
-   Data Analysis

## Steps

1.  Load dataset
2.  Analyze statistics
3.  Identify relationships between variables

## Learning Outcome

Understanding how data analysis helps in decision making.
